Python Object Oriented Programming Project - Rock Snake!

September 2022

As part of my final assessment for Python, I coded the snake game using Object Oriented Programming (OOP). This involved planning the game and working out the algorithm in detail before finally implementing the code. Drawio was used to create the diagrams for the initial planning stage.

Using Object Oriented Programming

In OOP, classes of objects are identified, given properties (attributes) and methods (functions). These are then called during the main game loop. My classes included Snake, Food, Sounds and Messages.

The 'secret' to the continuously moving snake!

The x and y variables are first used to draw the 'snake' to the screen. x_change moves the snake 20px in the direction specified by the user. Pygame KEYDOWN is used to detect keyboard input. x += x_change means that the value of x (coordinates of the current position) is added to or subtracted from the value of x_change (+ or - 20px) each time the main loop runs (5 times per second). The same is specified for the y axis, allowing the snake to move continously.

Extra work

Once I had solved some initial problems getting the snake to move using OOP, I had a lot of fun and decided to add a few extra features to make it look cool! 😎

Color Change

The current snake length is divided in two to find the halfway point. When this point reaches 50px from the border, the snake should turn red to show he is in danger!

Snake Speed

Pygame KEYDOWN is used to allow the user to increase or decrease the snake speed. This is done by increasing/decreasing the number of times the main loop runs per second each time the specified keys are pressed.

Background and effects

I used pygame mixer to add some winning sounds and rock n' roll vibes!

Coding a simple chat app using Fernet encryption

September 2022

One of my final Python projects involved creating a simple chat app, using Fernet to encrypt messages as they travel accross the network. This involved creating a client and a server, binding the server to a socket to allow it to listen for incoming connections and then creating a simple WHILE loop to allow communication to continue until someone typed 'quit'.

More about Fernet

Fernet is part of Python's cryptography library and provides symmetric encryption, meaning that the same cryptographic key is used for both encryption and decryption. Fernet produces a token based on the plaintext message entered, a timestamp and the key. The ciphertext will always be a multiple of the 128-bit block size. If the message entered is not 128 bits, fernet will use padding to fill up any space that has not been used.
